# YsbCms
## 技术栈
1. NodeJs
1. Express
1. React/Vue 
## 目录解析
1. api 为前端提供图片上传、json数据上传等api功能逻辑
1. bin 工程启动脚本
1. components 通用组件目录
1. docs 项目文档
1. images 该目录会被设置为共享目录，最终会被挂载到图片服务器。用户上传图片时，将图片保存到该目录下即可。
1. node_modules 依赖的第三方node模块
1. public 服务端负责渲染的页面工作目录（目前用不到）
1. routes express路由控制
1. src 生成专题页的前端代码工作目录
1. tools 通用工具目录
